Cara Mendownload File

Untuk membuka file ISO, “mount” (pasang) file ISO tersebut menjadi sebuah virtual drive di komputer Anda lalu lakukan instalasi dari sana. Untuk melakukan mounting bisa mempergunakan software Daemon Tools Lite.

Atau gunakan software Nero Burning ROM untuk “membakar” (mengubah) file ISO yang telah Anda download di atas menjadi bentuk Blu-Ray / DVD, lalu lakukan instalasi melalui Blu-Ray / DVD tersebut.

Jika Anda ingin mendownload versi terbaru, perlu untuk menggunakan software pendownload .torrent. Anda bisa install dulu software semacam uTorrent, BitTorrent, Free Download Manager, sebelum mengklik link download torrent di atas.
